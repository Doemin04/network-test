1. DNS & IP Addressing

How our domain resolves: When a user types doemin04.github.io into their browser, a DNS (Domain Name System) query is initiated to translate our human-readable domain into an IP address.

IP Addressing: Our domain currently resolves to four IPv4 addresses (such as 185.199.108.153) and four IPv6 addresses (such as 2606:50c0:8000::153).

2. Protocols

HTTP vs HTTPS: Our site is secured using HTTPS (Hypertext Transfer Protocol Secure), which ensures that all communication between the browser and the GitHub Pages server is encrypted using TLS/SSL.

Status Codes: When the page loads successfully, the server returns a 200 OK status code: the client's request was successfully received, understood, and processed.

Response Headers: The server also sends back response headers that provide metadata about the response, such as the content type (text/html) and security policies.
